# STempLog

Simple temperature log
        
O_o



## How to install

To install it just run the following commands in the terminal

    $ curl https://raw.githubusercontent.com/zYoplyd/STempLog/main/stemp > stemp

    $ chmod +x stemp

    $ sudo ln -s $PWD/stemp /usr/local/bin/stemp
